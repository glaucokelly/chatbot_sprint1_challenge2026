# chatbot_sprint1_challenge2026
Arquitetura e documentação técnica da Sprint 1 da disciplina de Prompt and Artificial Intelligence para o EV Challenge 2026 FIAP e GoodWe. Projeto de chatbot que utiliza processamento de linguagem natural NLP para traduzir terabytes de dados brutos em orientações diretas e gerenciais para operadores comerciais e lojistas.
